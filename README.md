# Pronosticador de Ventas para Apoyo a la Definición de Metas

## Objetivo
Desarrollar una herramienta de apoyo al análisis de ventas que permita generar pronósticos y establecer metas realistas en base a datos históricos.

## Contexto
Proyecto desarrollado como ejercicio aplicado de análisis de datos para apoyar la planificación comercial y la toma de decisiones mediante modelos de pronóstico.

## Metodología
Se implementaron y compararon distintos modelos de pronóstico:
- Regresión Lineal
- Regresión Polinómica
- Suavización Exponencial Simple
- Holt-Winters

Los modelos se evaluaron visualmente para identificar tendencias y escenarios futuros.

## Herramientas utilizadas
- Python
- Pandas
- NumPy
- Scikit-learn
- Statsmodels
- Matplotlib
- FPDF

## Resultados
- Generación de pronósticos de ventas.
- Comparación de tendencias entre modelos.
- Elaboración automática de un informe en PDF con gráficos y explicaciones.
- Apoyo en la definición de metas comerciales realistas.

## Uso
El script permite ingresar datos históricos de forma manual y genera:
- Pronósticos mediante distintos modelos
- Gráficos comparativos
- Un informe en PDF con resultados y explicaciones

## Ejemplo de informe
El archivo `ejemplo_informe_pronostico.pdf` contiene un informe generado a partir de datos ficticios utilizados únicamente con fines demostrativos y académicos.
No corresponde a información real de ninguna empresa.
